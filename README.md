# Galatasaray Fan Sitesi

Bu proje Galatasaray hakkında hazırlanmış basit bir fan sitesidir.  
HTML ve CSS kullanılarak oluşturulmuştur.

## 📌 İçerik

Sitede şu sayfalar bulunmaktadır:

- Ana Sayfa
- Kadro
- Maçlar
- Galeri
- Tarihçe

## 🎨 Özellikler

- Sarı kırmızı tema
- Oyuncu kartları
- Arka plan animasyonları
- Kupa dolabı tasarımı
- Konfeti efekti
- Pinterest galeri bağlantısı
